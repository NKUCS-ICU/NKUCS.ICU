# 贡献指南

感谢你对 NKUCS.ICU 项目的关注！我们欢迎一切形式的贡献。

## 如何贡献

### 方式一：评论区留言
在每个课程页面下方的评论区留言，维护者会将有价值的内容添加到页面中。

### 方式二：发送邮件
将内容发送至维护人员邮箱：[emanual20@foxmail.com](mailto:emanual20@foxmail.com)

### 方式三：Pull Request

1. **Fork** 本仓库
2. 创建你的特性分支：`git checkout -b feature/my-contribution`
3. 修改对应的 Markdown 文件
4. 确保 Markdown 格式正确
5. 提交更改：`git commit -m 'Add: 课程XX评价'`
6. 推送到分支：`git push origin feature/my-contribution`
7. 发起 **Pull Request**

## 内容规范

### 课程评价
- 写明**修读学期**（如 24Fall）
- 写明**授课教师**
- 署名（可使用 GitHub 用户名，匿名可署 Anonymous）
- 包含实际内容，避免纯粹吐槽

### 经验分享
- 写明**时间范围**（如 2024年暑期）
- 署名
- 提供具体、有参考价值的信息

### 文件格式
- 所有内容使用 **Markdown** 格式
- 上传资源文件请使用 **PDF** 格式，单文件不超过 **2MB**
- 图片请使用图床链接

## 行为准则
- 不得发表人身攻击、辱骂等言论
- 不得发表违法犯罪言论
- 尊重每一位贡献者和读者
